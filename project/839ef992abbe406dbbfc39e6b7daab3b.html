작업 내용
1) PC&모바일 상품 큐레이션 필터
💡
작업내용 - 노출 위치 : 상품 리스트 페이지(타일>크기별, 빅슬랩 전체 카테고리에서는 ‘크기’ 필터항목 미노출, 타일>색상별 카데고리에서는 ‘색상’ 필터항목 미노출) - 필터 노출 유무는 카테고리 넘버로 구분(004, 009 카테고리와 그 하위 카테고리) - 검색 결과 노출 시 노출 순서 유지 - 모든 필터항목 중복 선택 불가 - 가격은 box/ea 가격과 ㎡ 가격 중 한가지만 선택 가능 - 선택된 항목 스타일 변경, 우측에 취소 버튼(x)노출 - 전체 항목 선택 후 ‘선택한 조건으로 제품 찾기’버튼 클릭 시 결과 리스팅
💡
필터항목 - 원산지(이탈리아, 스페인, 중국, 대한민국, 기타국가) * 기준: 상품 옵션의 원산지 란에 입력된 값 (/ 중국 -> China 스페인 -> Spain 대한민국 -> Korea 이탈리아 -> Italy 기타 -> Various, 값이 없는 경우 /) - box/ea 가격(4만원 이하, 4~6만원, 6~8만원, 8~11만원, 11~15만원, 15만원 이상) - ㎡ 가격(3만원 이하, 3~5만원, 5~7만원, 7~10만원, 10만원 이상) * 기준: 판매가 - 디자인(스톤, 마블, 우드, 테라조, 콘크리트, 패턴, 모자이크, 3D, 기타디자인) * 기준: 검색 키워드 - 색상(화이트, 베이지, 그레이, 그린, 블루, 옐로우, 레드, 브라운, 기타색상) * 기준: 검색용 사이즈 옵션 셀렉트박스(새로 제작) - 크기(300mm 이하, 300x300mm, 300x600mm, 600x600mm, 600mm 이하, 900mm 이하, 600x1200mm, 1200mm 이하, 1200mm 초과, 빅슬랩 6t, 빅슬랩 12t) * 기준: 검색용 사이즈 옵션 셀렉트박스(새로 제작)
