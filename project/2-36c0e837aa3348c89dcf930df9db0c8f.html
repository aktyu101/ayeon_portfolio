  1. 맞춤 제작 a. ★메인페이지
  • 최초시안 2안 제공, 수정 4회 이내
  • 헤더, 풋터, 모바일 햄버거메뉴, 모바일 하단바 작업 포함
  • 팝업 프레임 1개 작업 (팝업 배너 디자인 제외) b. ★제품 구매 관련
  • 상품 리스트 : 카테고리별 공통 디자인, 검색 결과페이지 포함
  • 상품 상세 : 상세 디테일컷을 제외한 전체 레이아웃 c. ★기타 추가작업(스킨 셋팅 후 작업진행)
  • 마이페이지 메인 내 등급안내 영역 추가, 회원혜택 팝업 추가
  • 마이페이지 최근본상품 페이지 추가
  • 마이페이지 1:1문의 메뉴 추가 (1:1문의 게시판으로 이동) **1:1문의 게시판 확인 중
  • 미디어페이지 추가 (탭 3개 구분 추가, 섬네일 크기 및 간격 조정)
  • 로그인, 이벤트 페이지 타이틀 추가
  • 로그인 페이지 내 배너영역 1개 추가
  • 회원혜택&등급별혜택, 쇼핑몰 이용안내, 수출 및 제휴문의 (기존 컨텐츠 활용하여 제작)
  • PC 자주묻는 질문 내 고객센터 탭 추가 & MO 고객센터 페이지 내 탭 추가
  • 기타 유지보수 (디자인 레이아웃이 변경되지 않는 선의 간단한 작업)
  1. 템플릿 활용 제작 (타사 스킨 템플릿 - 모던 스킨 04 - 모던 (위브측에서 구매) https://ecudemo205431.cafe24.com/https://ecudemo205431.cafe24.com/m) a. 개별 디자인페이지
  • 브랜드소개(기존 나인위시스 디자인 활용하여 동일하게 제작 or 스킨 템플릿 활용) b. 로그인 관련
  • 회원 로그인, 아이디찾기, 비밀번호찾기, 비회원구매, 비회원 주문 c. 게시판 관련
  • 공지사항, 자주묻는질문, 1:1문의, 이벤트, 갤러리게시판 d. 기본페이지 / 마이페이지
  • 회원가입 관련 : 회원가입약관, 정보입력, 가입완료, 이용약관, 개인정보처리방침
  • 주문 관련 : 장바구니(체크한 상품만 가격 표시되도록 수정, 네이버페이 버튼 크기 조정), 주문서작성, 주문완료
  • 마이페이지 : 메인, 주문/배송조회, 관심상품, 적립금, 쿠폰, 내가쓴게시물, 배송지관리, 회원정보수정, 회원탈퇴
※ 컨텐츠 영역 작업 제외 : 기획전 페이지, 분류 카테고리별 타이틀배너. 게시판별 타이틀배너 등 (제작 요청 시 별도 비용 발생)
